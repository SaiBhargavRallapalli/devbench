# homebrew-devbench

Homebrew tap for [DevBench](https://www.devbench.co.in) — CLI and macOS desktop app.

```bash
brew tap SaiBhargavRallapalli/devbench
brew install devbench              # terminal CLI
brew install --cask devbench       # macOS desktop app
```

- App repo: [SaiBhargavRallapalli/devbench](https://github.com/SaiBhargavRallapalli/devbench)
- Releases / DMGs: [GitHub Releases](https://github.com/SaiBhargavRallapalli/devbench/releases)
- Download page: [devbench.co.in/download](https://www.devbench.co.in/download)

Formula and Cask files are updated automatically when a [Release workflow](https://github.com/SaiBhargavRallapalli/devbench/actions/workflows/release.yml) completes.
