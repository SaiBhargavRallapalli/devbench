# homebrew-devbench

Homebrew tap for [DevBench](https://www.devbench.co.in).

```bash
brew tap SaiBhargavRallapalli/devbench
brew install devbench              # terminal CLI (json-format, base64, yaml, …)
brew install --cask devbench       # macOS desktop app
```

Releases are updated automatically from [devbench](https://github.com/SaiBhargavRallapalli/devbench) tags.
