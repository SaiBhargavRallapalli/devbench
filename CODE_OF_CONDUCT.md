# Code of conduct

## Our pledge

We want DevBench to be a welcoming place for contributors and users. We adopt the **[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** (version 2.1) as our community standard. Please read the full text at the link above.

## Reporting

- For **conduct** concerns related to this open-source project (issues, pull requests, discussions), open a GitHub issue describing the situation (you may ask a maintainer to keep details private where the platform allows), or use the **[Contact](https://www.devbench.co.in/contact)** page on the website if you prefer email.
- For **security vulnerabilities**, use **[`SECURITY.md`](SECURITY.md)** — not a public issue.

Maintainers may warn, edit, or remove contributions that violate this code and may ban repeat offenders from the project spaces they control.
